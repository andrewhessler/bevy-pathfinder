# Paint By Algorithms (aka Bevy Pathfinder)

Pathfinding looked like a fun project. Figured I'd use it to learn Bevy.

## How to run
`cargo run`

```
WASD to move
J - to pulse pathfinding
```

Algorithms (Press Number to paint with)

```
1 - Djikstra (Default)
2 - AStar
3 - AggressiveStar
```

Build Mode (Left Click builds...)
```
Wall - R (Default)
End  - E
Right-Click - Remove
```
